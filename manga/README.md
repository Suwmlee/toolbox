
### 使用方法
```
usage: run.py [-h] [-t] [-f] [-tf] [-og] [-kg]

options:
  -h, --help       show this help message and exit
  -t, --test       测试模式
  -f, --force      强制执行变更
  -tf, --transfer  tachiyomi转移到komga
  -og, --organize  整理manga
  -kg, --komga     根据komga进行整理
```

#### transfer_manga:

移动 tachiyomi 下载目录内容到 komga 目录

#### organize_manga:

优化 文件夹结构 优化命名

#### organize_komga:

更新 作者和 星星
移动 星星 标记的文件到匹配星星的目录
